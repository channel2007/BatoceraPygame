Batocera模擬器上玩自製Pygame遊戲

# 遊戲介紹
![alt 太空侵略者](https://github.com/channel2007/BatoceraPygame/blob/master/images/SpaceInvaders-image.png "太空侵略者")
* 遊戲名稱：太空侵略者(彩色版/灰階版)(單人遊戲)
* 檔案名稱：SpaceInvaders、SpaceInvadersGrayScale
* 操作方式：鍵盤
  * 左右：移動飛機
  * 空白建：發射子彈、開始遊戲
  * 左邊Shift：加快飛機移動速度
  * Esc：離開遊戲
* 操作方式：搖桿
  * 左右：移動飛機
  * Ａ：發射子彈
  * Ｂ：加快飛機移動速度
